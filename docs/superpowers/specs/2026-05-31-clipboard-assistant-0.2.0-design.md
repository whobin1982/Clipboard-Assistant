# 剪贴板助手 0.2.0 设计

## 目标

0.2.0 聚焦两个体验提升：让剪贴板历史窗口可以用键盘快速选择和粘贴；让设置页能控制常用行为。这个版本不扩展到隐私黑名单、分类标签、云同步等较大功能。

## 用户体验

打开剪贴板历史窗口后，列表默认不选中任何记录。用户按一次下方向键后选中第一条，也就是最新记录；之后可以用上、下方向键移动选择。选中项使用系统原生列表选中效果，并需要在浅色和深色模式下都清楚可见。

按回车时，应用对当前选中记录执行“选择记录”动作。动作完成后是否关闭窗口由设置控制，默认关闭窗口。按 Esc 时关闭历史窗口。

选择记录的动作由设置控制。默认行为是把记录写入系统剪贴板，并自动粘贴到当前输入框；用户也可以在设置页切换为只写入系统剪贴板，不执行自动粘贴。

## 设置页范围

设置页在 0.2.0 中包含这些配置：

- 选择记录后是否自动粘贴，默认开启。
- 选择记录后是否关闭历史窗口，默认开启。
- Esc 是否关闭历史窗口，默认开启。
- 快捷键设置，默认 `⌥⌘V`。
- 开机启动。
- 历史保留天数，默认 30 天。

快捷键设置同时提供常用固定选项和自定义录入。固定选项为 `⌥⌘V`、`⌃⌘V`、`⇧⌘V`，默认 `⌥⌘V`，并保留已有快捷键服务的兼容行为。自定义快捷键需要保存到现有设置模型，并在修改后重新注册全局快捷键。

历史保留天数提供常用选项 7 天、30 天、90 天、永久，同时支持自定义天数。自定义天数应限制为正整数；永久保留需要能被存储层和清理逻辑明确识别。

## 架构调整

### 设置模型

`AppSettings` 增加 0.2.0 所需字段：

- `selectionAction`：自动粘贴或只复制。
- `closeWindowAfterSelection`：选择后是否关闭窗口。
- `escapeClosesWindow`：Esc 是否关闭窗口。
- `shortcutDefinition`：当前快捷键配置，支持固定选项和自定义组合键。
- 历史保留策略，支持固定天数和永久。

默认值保持面向普通用户的顺手体验：自动粘贴、选择后关闭、Esc 关闭、默认快捷键 `⌥⌘V`、保留 30 天。

### 历史窗口

历史窗口需要把键盘事件转成列表选择和执行动作。实现时优先保持现有 `SearchWindowPresenter` 负责窗口生命周期，`ClipboardPopupView` 负责界面状态和列表交互。

列表选择状态由视图层维护。窗口打开时选择状态为空；方向键第一次向下时选中第一条。回车只在存在选中记录时生效。记录列表刷新或搜索过滤变化后，如果当前选中记录不再存在，应清空选择，避免误粘贴。

### 选择记录动作

选择记录动作应复用现有复制和粘贴服务。自动粘贴模式下，流程保持为写入剪贴板、关闭窗口、激活原应用、发送粘贴快捷键。只复制模式下，只写入系统剪贴板，并根据设置决定是否关闭窗口，不发送粘贴快捷键。

来自剪贴板助手自身的写入仍然需要带标记，避免被监听器新增为历史记录。

### 设置页

设置页使用中文文案。设置项按使用频率分组：

- 粘贴行为：自动粘贴、选择后关闭、Esc 关闭。
- 快捷键：固定选项和自定义录入。
- 启动与历史：开机启动、保留天数。

设置变更需要即时生效。快捷键变更后重新注册；保留天数变更后立即触发一次清理，并继续沿用现有定期清理流程。

## 错误处理

自动粘贴失败时，记录仍应已经放入系统剪贴板，并在界面中显示简短中文错误提示。只复制模式通常不需要辅助功能权限；自动粘贴模式仍依赖辅助功能权限。

自定义快捷键如果录入失败、冲突或组合键不完整，应显示中文提示并不覆盖原快捷键。

自定义保留天数如果为空、不是数字或小于 1，应保留原设置并显示中文提示。

## 测试

新增或更新测试覆盖：

- 默认设置值符合 0.2.0 预期。
- 只复制模式不会发送粘贴快捷键。
- 自动粘贴模式继续发送粘贴快捷键。
- 选择后关闭窗口配置生效。
- Esc 关闭窗口配置生效。
- 历史保留天数支持固定选项、自定义正整数和永久。
- 快捷键固定选项和自定义配置能持久化并通知快捷键服务。
- 方向键选择和回车执行的核心逻辑可通过可测试的视图模型或控制器验证。

## 不做范围

0.2.0 不包含：

- 应用黑名单。
- 密码或验证码识别过滤。
- 分类、标签、复杂搜索。
- 云同步或多设备同步。
- 自动更新机制。
